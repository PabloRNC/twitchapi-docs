Options for generating an emote URL.

## Properties

| Property | Type | Description |
| :------ | :------ | :------ |
| `format?` | `"static"` \| `"animated"` | The format of the emote. |
| `scale?` | `"1.0"` \| `"2.0"` \| `"3.0"` | The scale of the emote. |
| `theme?` | `"light"` \| `"dark"` | The theme of the emote. |
